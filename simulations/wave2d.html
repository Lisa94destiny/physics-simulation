波的二维干涉原理

波的二维干涉是波动现象的一种重要表现形式。当两个相干波源在同一介质中同时发出波时，这些波会在空间中相遇并产生干涉。

在二维平面上，干涉条件如下：

相长干涉：r₂ - r₁ = nλ

n = 0, ±1, ±2, ...

相消干涉：r₂ - r₁ = (n+1/2)λ

n = 0, ±1, ±2, ...

其中，r₁和r₂分别是从两个波源到观察点的距离，λ是波长。在二维平面上，相长干涉点形成的是一系列双曲线，这些双曲线就是干涉的极大值线；相消干涉点也形成一系列双曲线，即干涉的极小值线（也称节线）。
